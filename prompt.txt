

Je suis développeur web et je construis un site vitrine sur mesure pour un client (bureau d’études / ingénierie BIM).

Contexte technique :

Stack simple PHP MVC (index.php + controllers + views)

Utilisation de Bootstrap 5 pour toute la structure (grid, responsive, spacing)

CSS personnalisé léger, propre, maintenable

Pas de framework JS lourd

Code clair, lisible, structuré comme un vrai projet pro

Contraintes importantes :

Le site doit être 100 % responsive, y compris très petits écrans

Toujours penser mobile-first

Pas d’éléments qui débordent, pas de contenu coupé

Design moderne, sobre, crédible (ingénierie / BIM / BTP)

Classes CSS bien nommées, cohérentes et réutilisables pour un back-office futur

Pas de noms de classes génériques ou “bazar”

Logique de travail :

On travaille section par section

Tu proposes du HTML + Bootstrap + CSS quand nécessaire

Tu expliques brièvement les choix quand c’est utile

Tu restes simple : pas d’usine à gaz

Pages déjà en place ou en cours :

Accueil (hero, services, à propos, valeurs)

Présentation (en cours de structuration)

Services

Contact (formulaire + coordonnées + carte)

Ce que j’attends de toi :

M’aider à concevoir et coder des sections modernes et impactantes

Toujours garder une logique professionnelle orientée client

Proposer des sections qui apportent une vraie valeur ajoutée

Être franc : si une idée ou un choix n’est pas pertinent, dis-le

Objectif final :
Un site vitrine propre, crédible, moderne, pensé comme un vrai produit,
prêt à évoluer plus tard avec un back-office (CMS, admin, contenus dynamiques).
