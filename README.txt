SITE VITRINE – NBSOLUTIONS

Description
-----------
Projet de site vitrine développé en PHP avec une architecture MVC classique.
Le projet sert de base simple et évolutive pour un site vitrine, avec la
possibilité d’ajouter un back-office par la suite.

Le header et le footer sont déjà en place, responsives et compatibles mobile.

Environnement
-------------
- Développement en local
- Serveur : XAMPP
- PHP + Bootstrap
- Architecture MVC (controllers / views / models)

Lancement du projet
-------------------
1. Placer le projet dans le dossier :
   xampp/htdocs/siteVitrine

2. Démarrer Apache depuis XAMPP

3. Accéder au site via le navigateur :
   http://localhost/siteVitrine/index.php?page=accueil

Structure du projet
-------------------
siteVitrine/
- controllers/   (contrôleurs PHP)
- views/         (vues : header, footer, pages)
- models/        (modèles, à compléter)
- assets/        (CSS, images, JS)
- config/        (configuration)
- index.php      (point d’entrée)

État actuel
